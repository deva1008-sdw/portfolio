Project Concept

Aggressive OS battery configurations terminate background geofetch processes. SafeTracker solves this using a persistent foreground service widget with an offline key-value storage queue buffer.

Technology Choices

React Native (Expo)
AsyncStorage Offline Cache
Leaflet.js Web Maps
Node.js Express backend

Background Tasks

Registers location fetching structures within Expo's `TaskManager` module, running as a permanent status bar item on user confirmation. 

// Location trigger
TaskManager.defineTask(LOCATION_TASK_NAME, ({ data }) => {
    const { locations } = data;
    // Buffer coordinates to AsyncStorage
});
