Project Overview

Recon requires executing repetitive discovery utilities. BugBounty-AI automates this by chaining discovery binaries (`subfinder`, `httpx`, `katana`) inside background task workers and filtering logging noise using OpenAI APIs.

Technology Choices

FastAPI & Python (Web Servers)
Celery & Redis (Task Brokers)
Go-based CLI Utilities (Discovery Engines)
PostgreSQL (Scan Logs)

Task Orchestration

Workers run subprocess commands, tracking progress states, and writing results directly to relational PostgreSQL tables. 

# Worker trigger snippet
cmd = ["subfinder", "-d", domain, "-silent"]
result = subprocess.run(cmd, capture_output=True, text=True)
hosts = result.stdout.splitlines()
