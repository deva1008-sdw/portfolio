Project Concept

Flat scanners analyze package lockfiles but fail to check if code paths actually execute vulnerabilities. CNS is a "Code Digital Twin" that parses syntax trees, maps calls to Neo4j Graph DB, and flags runtime CVE exposures via Gemini AI.

Tech Stack Choice

Neo4j (Graph Relationship DB) tree-sitter AST (Syntax Traverser) Gemini 1.5 Flash (Context Risk Auditor) FastAPI (Python REST Engine)

Core Engineering

Ingestion processes parse source codes into files, imports, classes, and calls. The compiler creates relational mappings dynamically, queryable via Cypher loops. 

// Ingesting call maps
MERGE (f:File {path: $filePath})
MERGE (fn:Function {name: $funcName})
MERGE (f)-[:CONTAINS]->(fn)
