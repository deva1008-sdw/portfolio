1. Problem Statement & Scope

In security compliance training, showing network exploit behaviors (like DDoS saturation or credential brute-forcing) is highly effective. However, launching actual exploit scripts against live network systems in classroom environments is extremely risky and can lead to infrastructure crashes.

There is a critical need for a **safe, simulated sandboxed environment** where students and administrators can trigger network attacks and view real-time graphical impact tables, without exposing any actual hardware assets to threat vectors.

Project Goal: Build a secure, educational cybersecurity platform featuring an interactive React dashboard that executes controlled, simulated network attacks inside backend python threads, streaming packet rates via WebSockets and exporting printable PDF audit logs.

2. Ingestion & Logging Pipeline

Core Engineering Layers:

  • FastAPI Attack Control: Manages simulation states. Spawns, pauses, and terminates attack thread loops safely based on UI triggers.
  • Simulation Loop Workers: Execute mocked socket loops in isolated threads. Generates simulated packet records and validates dictionary passwords against lists like `rockyou.txt`.
  • Report Compilation System: Gathers database logs and parses them into table components. Exports incidents to printable PDF files using template generators.
  • React Analytics Dashboard: Ingests backend WebSocket metrics and plots packet frequency, host response latency, and system load lines dynamically.

4. Code Deep Dive: DDoS Mock Loop

Below is the Python thread loop used to simulate DDoS packet traffic rates and stream telemetry: 

# Python Thread class simulating DDoS packet rates
import threading
import time
import random

class DdosSimulator(threading.Thread):
    def __init__(self, target_ip: str, pps_rate: int, callback):
        super().__init__()
        self.target_ip = target_ip
        self.pps_rate = pps_rate
        self.callback = callback
        self._stop_event = threading.Event()
        
    def stop(self):
        self._stop_event.set()
        
    def run(self):
        while not self._stop_event.is_set():
            # Calculate mock packet variables
            packets_sent = self.pps_rate + random.randint(-50, 50)
            bytes_sent = packets_sent * random.randint(64, 1500)
            
            # Send metric details to socket callback
            self.callback({
                "target": self.target_ip,
                "packets_rate": packets_sent,
                "bytes_rate": bytes_sent,
                "timestamp": time.time()
            })
            time.sleep(1.0)
